# Communication Agent

Agent for professional workplace communication using tactical empathy techniques.

## Overview

This agent helps craft effective workplace communications using Chris Voss "Never Split The Difference" techniques - curious, open, collaborative tone that builds rapport and achieves outcomes through understanding.

## Agent

| Agent | Purpose |
|-------|---------|
| [communication](communication.md) | Writes and reviews workplace communications |

## Capabilities

The consolidated communication agent handles:

### Writing
- Negotiation emails and messages
- Slack team announcements
- Requests and proposals
- Difficult conversations
- Pushback communications

### Reviewing
- Tone and effectiveness check
- Tactical empathy validation
- Clarity of request/position
- Actionable next steps

## Core Techniques

### Tactical Empathy
Understand their perspective before presenting yours. Demonstrate you've heard them.

### Labeling
Name their emotions/position: "It seems like...", "It sounds like..."

### Calibrated Questions
Open-ended questions starting with "How" or "What" (never accusatory "Why").

### Accusation Audit
Preemptively address their concerns before they raise them.

## When to Use

**Use for:**
- Requesting resources from leadership
- Proposing changes to stakeholders
- Deadline extension requests
- Delivering difficult feedback
- Pushing back on requirements
- Cross-team negotiations
- Team announcements (Slack)
- Project status updates
- Requests for help or input

**Do NOT use for:**
- Technical documentation (use tech-writer)
- Code comments
- Formal reports

## Workflow

```
1. Describe the situation and goal
2. Agent drafts communication
3. Review draft and notes
4. Optionally request review of your draft
5. Refine and send
```
